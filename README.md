# Profile

**My Portfolio - Abhijit Kumar Jha**

This is a personal portfolio web application built using React, Vite, and TypeScript.

## Tech Stack

- **Framework**: [React](https://react.dev/) (v19)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Internationalization**: [i18next](https://www.i18next.com/)
- **Styling**: Sass / SCSS
- **Testing**: [Jest](https://jestjs.io/) & [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
- **Linting & Code Quality**: [ESLint](https://eslint.org/) & [Stylelint](https://stylelint.io/)

## Prerequisites

- **Node.js**: >= 24.12.0 (Check `.nvmrc` for details)
- **npm**: Node package manager

## Available Scripts

In the project directory, you can run:

### `npm run dev`
Runs the app in the development mode using Vite. Open [http://localhost:5173](http://localhost:5173) (default port) to view it in your browser.

### `npm run dev-exposed`
Runs the app in development mode and exposes it to your local network.

### `npm run build`
Builds the app for production to the `dist` folder. It correctly bundles React in production mode and optimizes the build for the best performance. Before building, it also runs the TypeScript compiler.

### `npm run preview`
Locally preview the production build after running `npm run build`.

### `npm run test`
Launches the test runner (Jest) in standard mode.

### `npm run test-watch`
Launches the test runner in the interactive watch mode.

### `npm run test-filter <pattern>`
Run specific tests matching the provided pattern.

### `npm run lint`
Lints the TypeScript/JavaScript files using ESLint.

### `npm run lint-fix`
Lints the TypeScript/JavaScript files and automatically fixes found issues where possible.

### `npm run lint-style`
Lints the SCSS/CSS files using Stylelint.

### `npm run lint-style-fix`
Lints the SCSS/CSS files and automatically fixes style issues.

### `npm run clean`
Cleans the npm cache and removes the `node_modules` directory.
